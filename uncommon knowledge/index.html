LOGO

Original logo with black text must be used only on a bright background:

  • Blue
  • Green
  • Orange
  • Pink
  • Yellow

Do

  • - Keep enough space around the logo so that it doesn't appear cluttered
  • - Keep the logo to the original ratio
  • - Keep icon colours original

Original logo with white text must be used only on a dulled background:

  • Red
  • Blue
  • Purple
  • Black

Don't

  • - Alter the reationship of the elements in anyway
  • - Change the font or alter the spacing between the letters
  • - Reproduce with a font colour other than black or white
  • - Render to look 3D
  • - Place on a busy or multicoloured background
  • - Use as a pattern

COLOUR

All Colour within the logo must remain the same, only the text around the icon may change. Text around th icon must only change from black to white, and white to black when relevant i.e to suit the appropriate background.

All items i.e notepads, business cards must follow them same rules as mentioned above.

Book covers must keep assigned colours and colours may only change brightness, Darkened colour for a Hard Copy of the book and Brightened colour for a Digital Copy. On Hard Copies White text only must be used and on Digital Copies Off-White/Very Light Grey text only to be used.

ICONS

  • Icons must not be rotated or manipulated in anyway
  • Must only be used for relevant topics
  • Don't change official colou scheme, unless changing entire set to the same colour for the relevant theme
  • Must not be associated with anything else other than designated topic
  • Must not be used to create a pattern
